# JavaScript Todo App

A simple Todo application built with HTML, CSS, and Vanilla JavaScript to practice core JavaScript concepts and DOM manipulation.

## Features

- Add tasks
- Delete tasks
- Mark tasks as complete
- Save tasks using Local Storage
- Filter tasks by status

## Tech Stack

- HTML
- CSS
- JavaScript (ES6+)

## Project Structure

```text
01-JavaScript-Todo-App/
├── index.html
├── style.css
└── script.js
```

## Learning Goals

- DOM Manipulation
- Event Handling
- Arrays & Objects
- Local Storage
- JavaScript Fundamentals
